# Sentiment Preprocessing and Visualization

## Week 2 Task

Implemented preprocessing techniques for sentiment analysis, feature engineering methods, and data visualization to prepare product review data for machine learning models.

---

## Objective

To clean and preprocess textual review data, extract meaningful numerical features, and visualize sentiment-related patterns in the dataset.

---

## Dataset

Product Reviews Dataset

### Dataset Features

- reviewText : Customer review text
- overall : Product rating
- summary : Review summary
- sentiment : Sentiment label

---

- Python
- Pandas

- Scikit-Learn
- Matplotlib
- Seaborn
- WordCloud

- GitHub

---

## Features Implemented

### Text Cleaning
- Converted text to lowercase
- Removed punctuation marks
- Removed stop words
- Tokenized review text
- Removed extra spaces

### Sentiment Feature Engineering
- Applied TF-IDF Vectorization
- Generated numerical representations of text
- Prepared training and testing datasets

### Visualization
- Sentiment Distribution Plot
- Word Frequency Analysis
- Word Cloud Generation

---

## Steps Performed

### 1. Data Loading
Loaded the product review dataset.

### 2. Text Cleaning
Performed preprocessing to remove noise from textual data.

### 3. Feature Engineering
Used TF-IDF Vectorizer to convert reviews into feature vectors.

### 4. Visualization
Created plots to understand sentiment distribution and frequently occurring words.

---

## Results

Successfully preprocessed textual review data, extracted sentiment-related features, and generated visualizations for exploratory analysis.
